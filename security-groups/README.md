
# Security Groups Lab

## Objectives

- Create inbound rules.
- Create outbound rules.
- Restrict traffic.
- Understand common ports.

## Common Ports

| Port | Service |
|--------|---------|
| 22 | SSH |
| 80 | HTTP |
| 443 | HTTPS |
| 3389 | RDP |

## Notes

Security groups act as stateful virtual firewalls that control traffic to AWS resources.
