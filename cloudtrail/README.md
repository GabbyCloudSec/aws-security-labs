## CloudTrail Lab

### Objectives

- Enable CloudTrail
- Monitor account activity
- Review API events
- Understand logging and auditing

### Benefits

`Visibility` `Auditing` `Compliance` `Incident Investigation`

### Notes

CloudTrail records activity across AWS accounts and helps support security monitoring — this is the same log-analysis skill applied in [aws-iam-investigation](https://github.com/GabbyCloudSec/aws-iam-investigation) to trace a real (simulated) incident from alert to resolution.
