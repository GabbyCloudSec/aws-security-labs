Hands-on AWS security labs focused on identity security, logging, and cloud infrastructure protection.

This repository simulates real-world cloud security scenarios involving misconfigurations, access control risks, and monitoring analysis using AWS services.

## Focus Areas
- IAM security and least privilege access
- CloudTrail log analysis and monitoring
- S3 security and access control
- EC2 security configuration review
- Cloud security best practices

## Skills Developed
- AWS security fundamentals
- Security risk identification
- Cloud monitoring and investigation
- Incident-style thinking and analysis
